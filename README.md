# Compiladores-2024.2

Este repositório, **Compiladores-2024.2**, tem como objetivo armazenar todos os projetos e atividades desenvolvidos durante a disciplina de Compiladores no semestre 2024.2. 

## O que é um Compilador?

Um compilador é um programa de computador que traduz código fonte escrito em uma linguagem de programação de alto nível para uma linguagem de máquina, que pode ser executada diretamente pelo processador. 
O objetivo principal de um compilador é permitir que programas escritos em linguagens de alto nível sejam convertidos em um formato que o computador possa entender e executar.

### Tópicos Abordados:

1. **Análise Léxica**:
   - O processo de dividir o código fonte em unidades léxicas chamadas tokens.

2. **Análise Sintática**:
   - Análise da estrutura gramatical do código fonte, verificando se a sequência de tokens está de acordo com a gramática da linguagem de programação.

3. **Análise Semântica**:
   - A verificação do significado dos tokens e das estruturas gramaticais para garantir que o código fonte faz sentido de acordo com as regras da linguagem e o contexto em que está sendo usado.
